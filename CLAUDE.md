# Правила работы с проектом

## О проекте

Личный сайт Константина Петрицына — наставника по счастью и вайбкодера. Точка входа для новых клиентов через Telegram. Проект растёт постепенно: сейчас это визитка, дальше — портфолио и форма записи на консультацию.

Текущий файл: `vizitka.html`
Репозиторий: https://github.com/Konst2026/vizitka

## Структура файлов

Все страницы делаем в одном HTML-файле. Никаких внешних библиотек, фреймворков или CDN — только чистый HTML, CSS и JS.

## Дизайн

Используем текущий дизайн как основу. Не меняй цветовую схему, шрифты и общий стиль без явного запроса.

**Цветовая палитра:**
- Фон: `#f5ede0` (светло-бежевый)
- Основной текст: `#2c1a0e`
- Акцент (кнопки): `#ff6a00` (ярко-оранжевый)
- Золотистые тона: `#b8860b`, `#daa520`, `#ffd700`
- Карточка: `rgba(255,248,235,.85)` с `backdrop-filter: blur`

**Типографика:** Segoe UI / system-ui. Имя — анимированный золотой градиент через `-webkit-background-clip: text`.

**Компоненты:**
- Карточка: `max-width: 420px`, `border-radius: 28px`, полупрозрачный фон, мягкие тени
- Кнопки: `border-radius: 16px`, анимированный градиент, hover с подъёмом через `translateY`
- Соцсети: квадратные кнопки `52×52px`, `border-radius: 16px`
- Аватар: круглое фото с вращающейся оранжево-золотой рамкой-градиентом
- Toast-уведомление: фиксированное внизу экрана, появляется снизу

**Анимации:** используем CSS-анимации (`@keyframes`), никакого JS для анимаций внешнего вида.

## Адаптивная вёрстка

Всё должно хорошо выглядеть на мобильных устройствах. Проверяй вёрстку на экранах от 320px. Используй гибкие единицы (%, vw, em, rem), flexbox или grid. Медиазапрос для мобильных: `@media (max-width: 420px)`.

## Стиль текста

Простой и дружеский язык. Без канцелярита, сложных конструкций и официоза. Пишем так, как говорим.

## Контент и тематика

Тематика сайта — счастье, саморазвитие, коучинг, AI, вайбкодинг. Основной призыв к действию — написать в Telegram (`https://t.me/+jaTkGvqx6C5jODk5`). Все ссылки на Telegram открываются в `target="_blank" rel="noopener"`.
