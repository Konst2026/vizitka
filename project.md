# Личный сайт Константина Петрицына


## О проекте

Личный сайт Константина Петрицына — наставника по счастью и вайбкодера.
Сайт служит точкой входа для новых клиентов: знакомит с автором, его подходом и даёт возможность записаться на консультацию.
Проект развивается постепенно — от визитки до полноценного персонального сайта.

## Для кого

Для людей, которые хотят найти наставника по личностному развитию и счастью.
Основной канал привлечения — Telegram.

| Страница | Описание |
|----------|----------|

## Дизайн

- **Цветовая схема:** светло-бежевые тона, тёплые золотистые акценты
- **Акцентный цвет:** ярко-оранжевый (`#ff6a00`) — кнопки и активные элементы
- **Типографика:** Segoe UI / system-ui, имя — анимированный золотой градиент
- **Фото:** большое круглое фото автора с оранжевой рамкой-градиентом
- **Стиль карточек:** скруглённые углы, полупрозрачный фон, мягкие тени
